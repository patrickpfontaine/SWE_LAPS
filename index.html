JavaScript Properties and Functions

.length
This property returns a number of elements in an array or the number of characters in a string

.substring(start,end)
Extracts part of the string but does not accept negative indices

.slice(start,end)
Extracts part of the array or string

.split(separator)
Convert a string into an array based on a separator

Array.from()
Can create an array from an iterable object, like a string

eval()
Executes a string of JavaScript code in the current context
